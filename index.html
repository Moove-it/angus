Why Angus

Performance

Information about this charts

We made this charts using Apache Benchmark. We created one project per framework (Grape, Rails API, Sinatra and Angus) that provides a simple service 'get users' which return a list of users in JSON format with the default and basic configuration. In the Ruby environment we use Unicorn with defult configuration. With jRuby we use Mizuno. The test consisted of 5000 requests executed with a concurrency level of 100.

Tools

Remote client
angus-remote gem provides a way to interact with a service created with Angus.

Authentication
angus-authentication gem provides an authentication layer to Angus services.

More to come
We are creating some new features and gems to enrich Angus.

Using Angus

$ gem install angus
$ angus demo
$ cd demo
$ bundle install
$ angus server

Ready! Go to
